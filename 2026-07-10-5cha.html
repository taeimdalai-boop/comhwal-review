인쇄 설정은 3단계 루틴으로

①보기 탭 → 페이지 나누기 미리 보기로 영역 확인 → ②Ctrl+P로 전체 그림 확인 → ③페이지 설정에서 여백·반복할 행·배율 등 세부 조정. 이 순서를 루틴으로 만들면 외우지 않아도 몸이 기억합니다.

암기: "보기 바꾸고 → Ctrl+P로 보고 → 페이지 설정에서 고친다"

정렬은 엑셀이 표를 알아봐야 된다

표 위에 "단위: 천원" 같은 문구가 붙어 있으면 엑셀이 데이터 범위를 인식하지 못해 정렬이 제대로 안 됩니다. 이럴 땐 범위를 직접 선택(B3에서 Ctrl+Shift+방향키)해서 알려줘야 합니다.

Ctrl+A를 눌러보면 엑셀이 표를 인식하는지 바로 확인 가능

다중 기준 정렬 + 셀 색 정렬

정렬 대화상자에서 기준 추가로 2차·3차 기준을 넣을 수 있고, 셀 값뿐 아니라 셀 색·글꼴 색으로도 정렬할 수 있습니다. 조합·회원사처럼 가나다순이 아닌 순서는 사용자 지정 목록으로 정렬합니다.

사용자 지정 목록은 입력 후 반드시 '추가' 버튼을 눌러야 저장됨

자동 필터 = 숨기기, 지우기 아님

자동 필터는 조건에 맞지 않는 행을 삭제하는 게 아니라 숨깁니다. 필터가 걸린 상태에서 복사하면 화면에 보이는 데이터만 복사됩니다. 여러 필드에 조건을 걸면 AND로 동작합니다.

서로 다른 필드의 OR 조건은 자동 필터로 안 됨 → 고급 필터

고급 필터: 같은 줄 AND, 다른 줄 OR

고급 필터는 조건 목록을 먼저 만들어야 하고, 필드명은 원본 표와 똑같이 적어야 합니다. 조건을 같은 줄에 쓰면 AND(그리고), 다른 줄에 쓰면 OR(또는)입니다.

★시험 단골: 조건 배치만 보고 AND인지 OR인지 판단하기

함수 기초: =함수명(인수)

함수는 =로 시작하고 함수명 뒤 괄호에 인수를 넣습니다. SUM·AVERAGE·MAX·MIN, 개수 세는 COUNT(숫자)·COUNTA(비어있지 않은 셀)·COUNTBLANK(빈 셀)를 구분해야 합니다.

함수 자동완성 목록에서 선택은 Enter가 아니라 Tab

조건부 집계: SUMIF·AVERAGEIF·COUNTIF

SUMIF(조건 검사 범위, 조건, 합계 범위) 순서입니다. AVERAGEIF도 같은 구조이고, COUNTIF는 (범위, 조건)만 씁니다. 채우기 핸들로 복사할 땐 범위를 절대 참조로 고정해야 합니다.

암기: "어디서 찾을지 → 뭘 찾을지 → 뭘 더할지"

참조: 상대·절대·혼합, F4로 순환

채우기 핸들을 아래로 내리면 행 번호가, 오른쪽으로 끌면 열 문자가 변합니다. $C$7은 절대(둘 다 고정), $B8은 열 고정, C$7은 행 고정. F4를 누르면 네 가지 형태가 순환됩니다.

고정하고 싶은 것 앞에 $ — "달러는 자물쇠"

💡개념 다시 보기

인쇄 설정 3단계 루틴 (복습 + 심화)

1단계 보기 탭 → 페이지 나누기 미리 보기. 인쇄 영역에 쓸데없는 내용(단가표·메모 등)이 포함됐는지 확인하고, 파란 경계선을 드래그해 조정합니다.
2단계 Ctrl+P로 인쇄 미리 보기. 여백·치우침·머리글 유무·페이지 번호 필요 여부를 끝 페이지까지 쭉 확인하고, 고칠 목록을 메모합니다.
3단계 페이지 설정 대화상자에서 하나씩 해결: 여백 탭 가로/세로 가운데 맞춤, 시트 탭 반복할 행, 머리글/바닥글, 배율 조정.
주의 인쇄 미리 보기 상태에서는 반복할 행을 지정할 수 없습니다. ESC로 빠져나온 뒤 페이지 레이아웃 탭 → 인쇄 제목에서 지정하세요.
레코드 균등 배치 전체 레코드 수(예: 224개)를 세고(Ctrl+Shift+↓) 목표 페이지 수로 나눠(224÷7=32) 페이지 나누기 선을 32개씩 끊어지게 옮깁니다. 상태 표시줄의 개수를 보면서 세면 편합니다.
시험 예시: "내용이 왼쪽으로 치우쳐 있을 때 빈 열을 삽입한다"는 틀린 방법 — 여백 탭의 가로 가운데 맞춤을 써야 합니다.

정렬 — 엑셀이 데이터베이스를 인식해야 한다

정상 케이스 표가 깔끔하면 셀 하나만 클릭하고 데이터 탭 → 오름차순/내림차순을 눌러도 알아서 전체가 정렬됩니다.
문제 케이스 표 위에 "단위: 1000원" 같은 문구가 셀에 입력돼 있으면 엑셀이 어디까지가 표인지 인식하지 못합니다. Ctrl+A를 눌러 표 전체가 선택되는지로 인식 여부를 확인할 수 있습니다.
해결법 범위를 직접 선택해 알려줍니다: 필드명 첫 셀(예: B3) 클릭 → Ctrl+Shift+→ → Ctrl+Shift+↓ 그리고 정렬 실행.
다중 기준 정렬 대화상자에서 '기준 추가'로 1차(부서명 오름차순) → 2차(중형차의 셀 색이 파란색 계열 먼저) 같은 복합 정렬이 가능합니다.
사용자 지정 목록 조합→회원사→비회원사처럼 가나다순이 아닌 순서로 정렬하려면 사용자 지정 목록에 순서를 입력하고 반드시 '추가' 버튼을 누른 뒤 사용합니다.
합계 행 주의 표 아래 합계(요약) 행이 붙어 있으면 정렬 범위에서 제외해야 합니다. 그러지 않으면 합계 행이 데이터 사이에 섞여 버립니다.
시험 예시: 문제 지문은 끊어 읽으세요 — "부서명을 기준으로 오름차순, 동일한 부서일 경우 중형차의 셀 색이 위에 오도록" = 기준 2개.

자동 필터 — 조건에 맞는 것만 보이게

켜기/끄기 표 안에 셀 포인터를 두고 데이터 탭 → 필터. 필드명에 ▼ 단추가 생깁니다.
동작 원리 조건에 맞지 않는 행은 삭제되는 게 아니라 숨겨집니다. 행 번호가 파란색으로 띄엄띄엄 보이면 필터가 걸린 상태입니다.
숫자 필터 크거나 같음(이상), 보다 큼(초과), 보다 작음(미만), 크거나 같고~작거나 같음(사이) 등 조건을 지정할 수 있습니다.
복사 특징 필터가 걸린 상태에서 복사하면 화면에 보이는 데이터만 복사됩니다. 필요한 데이터만 뽑아 옮길 때 유용합니다.
한계 여러 필드에 조건을 걸면 AND(모두 만족)로만 동작합니다. "부서가 영업1부이거나 직급이 과장"처럼 서로 다른 필드의 OR 조건은 자동 필터로 안 됩니다.
시험 예시: "자동 필터를 적용하면 조건에 맞지 않는 데이터는 삭제된다" — 틀린 설명, 숨겨질 뿐입니다.

고급 필터 — 조건 목록을 먼저 만든다

순서 ①빈 곳에 조건 목록 작성(필드명 + 조건) → ②데이터 탭 → 고급 → ③목록 범위·조건 범위·결과 위치 지정.
필드명 일치 조건 목록의 필드명은 원본 표의 필드명과 완전히 같아야 합니다. 오타가 나면 조건이 무시됩니다.
AND vs OR 조건을 같은 줄에 쓰면 AND("그리고"), 다른 줄에 쓰면 OR("또는")입니다. 이게 고급 필터의 핵심입니다.
다른 위치에 결과 복사 '다른 장소에 복사'를 선택하면 원본은 그대로 두고 결과만 추출됩니다.
일부 필드만 추출 결과 위치에 추출하고 싶은 필드명들을 미리 복사해 두고, '복사 위치'에 그 필드명 범위 전체를 지정합니다. 시작 셀 하나만 지정하면 안 됩니다.
비교 연산자 >=(이상), >(초과), <=(이하), <(미만), <>(같지 않다). =>나 =<처럼 순서를 바꿔 쓰면 안 됩니다.
시험 예시: 조건 목록에서 "영업부 / >=3000000"이 같은 줄 → 영업부이면서 급여 300만 이상(AND). 줄이 다르면 OR.

함수 기초 — 구조와 입력 요령

기본 구조 =함수명(인수1, 인수2, …). 반드시 =로 시작하고, 인수의 개수와 순서는 함수마다 정해져 있습니다.
입력 요령 =su까지 치면 자동완성 목록이 뜹니다. 여기서 Enter가 아니라 Tab을 눌러야 함수가 선택되며 괄호까지 자동으로 입력됩니다.
기본 5총사 SUM(합계), AVERAGE(평균), MAX(최댓값), MIN(최솟값), 그리고 개수 3형제.
개수 3형제 구분 COUNT는 숫자가 들어 있는 셀만, COUNTA는 비어 있지 않은 모든 셀, COUNTBLANK는 빈 셀만 셉니다.
조건부 집계 SUMIF(조건 검사 범위, 조건, 합계 범위) / AVERAGEIF(조건 범위, 조건, 평균 범위) / COUNTIF(범위, 조건). "어디서 찾을지 → 뭘 찾을지 → 뭘 계산할지" 순서로 기억하세요.
확인 도구 Ctrl+~(수식 표시), F9(수식 일부를 계산값으로 확인), F2(셀 수식 편집), fx 버튼(함수 인수 대화상자).
시험 예시: SUMIF의 세 번째 인수(합계 범위)를 빼먹거나, 조건 범위와 합계 범위를 서로 바꿔 쓰는 문제가 단골로 출제됩니다.

참조 — 상대·절대·혼합, 그리고 F4

참조란 셀 주소를 수식에 써서 그 셀의 값을 가져다 쓰는 것입니다. 값이 바뀌면 수식 결과도 자동으로 바뀝니다.
상대 참조 C7처럼 그냥 쓰면 채우기 핸들로 복사할 때 주소가 따라 움직입니다. 아래로 내리면 행 번호가, 오른쪽으로 끌면 열 문자가 변합니다.
절대 참조 $C$7 — 열과 행 모두 고정. 어디로 복사해도 항상 같은 셀을 봅니다.
혼합 참조 $B8은 열만 고정(B는 안 변하고 8은 변함), C$7은 행만 고정(C는 변하고 7은 안 변함).
F4 키 주소를 입력한 뒤 F4를 누르면 C7 → $C$7 → C$7 → $C7 순으로 순환됩니다.
실전 원칙 SUMIF·COUNTIF의 조건 범위·합계 범위처럼 "모든 행이 같은 범위를 봐야 하는 것"은 절대 참조로 고정하고, 조건처럼 행마다 달라져야 하는 것은 상대 참조로 둡니다.
시험 예시: 채우기 핸들로 아래로 복사했더니 결과가 이상하다 → 범위가 절대 참조로 고정되지 않아 함께 밀려 내려간 경우가 대부분입니다.

🚨자주 하는 실수

⭐ 가장 많이 틀린 부분

고급 필터에서 AND와 OR의 줄 배치를 반대로 써요

"영업부이면서 300만 이상"(AND)인데 조건을 두 줄에 나눠 쓰거나, "영업부이거나 300만 이상"(OR)인데 한 줄에 쓰는 실수가 가장 많습니다.

"같은 줄 = 그리고(AND), 다른 줄 = 또는(OR)" — 이 한 문장만 확실히 외우세요.

자주 하는 실수

표 위 "단위: 천원" 문구를 그대로 두고 정렬해요

표 위에 문구가 셀로 입력돼 있으면 엑셀이 데이터 범위를 잘못 인식해 정렬이 제대로 되지 않거나 엉뚱한 범위가 선택됩니다.

범위를 직접 선택(첫 필드명 셀에서 Ctrl+Shift+방향키)해서 엑셀에게 표를 알려주세요.

자주 하는 실수

채우기 핸들 복사에서 범위를 고정하지 않아요

SUMIF·COUNTIF를 아래로 복사할 때 조건 범위·합계 범위를 절대 참조($)로 고정하지 않으면 범위가 한 칸씩 밀려 내려가 결과가 틀어집니다.

범위를 지정한 직후 F4를 눌러 $를 붙이는 습관을 들이세요.

자주 하는 실수

비교 연산자를 =>, =< 로 거꾸로 써요

이상은 >=, 이하는 <=입니다. 등호를 앞에 쓰면 엑셀이 인식하지 못합니다. "같지 않다"는 <>입니다.

"부등호 먼저, 등호 나중" — >=, <= 순서를 소리 내어 외우세요.
