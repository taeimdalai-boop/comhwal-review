1과목 ① 한글 Windows 10 사용법 162문항 · 비중 1위

바로 가기 키 (단축키)

키기능
■(Windows)+D바탕 화면 표시/숨기기
■+E파일 탐색기 열기
■+L화면 잠금(사용자 전환)
■+I설정 열기
■+Tab작업 보기(타임라인·가상 데스크톱)
Alt+Tab실행 중인 앱 전환
Alt+F4현재 앱 종료 / 바탕화면에서 Windows 종료 대화상자
Ctrl+Shift+Esc작업 관리자 바로 열기
F2 / F3 / F5이름 바꾸기 / 검색 / 새로 고침
Shift+Delete휴지통 거치지 않고 완전 삭제
PrintScreen / Alt+PrintScreen전체 화면 / 활성 창만 클립보드 복사

파일·폴더 관리

  • 같은 드라이브에서 드래그 = 이동, 다른 드라이브로 드래그 = 복사. (Ctrl 누르면 항상 복사, Shift 누르면 항상 이동)
  • 파일 이름에 사용할 수 없는 문자: \ / : * ? " < > |
  • 휴지통에 보관되지 않는 경우: USB·네트워크 드라이브에서 삭제, Shift+Delete, 휴지통 크기 0%, 휴지통보다 큰 파일 → 즉시 완전 삭제되어 복원 불가
  • 바로 가기 아이콘(.lnk): 원본을 가리키는 포인터. 삭제해도 원본은 안 지워짐. 하나의 원본에 여러 개 생성 가능, 아이콘 왼쪽 아래 화살표 표시. Ctrl+Shift 누른 채 드래그로 생성. 원본 삭제 시 실행 불가
  • 파일·폴더 이름: 255자 이내, 공백 포함 가능. 같은 폴더 안에 같은 이름 불가
  • 작업 표시줄: 크기는 화면의 1/2까지 조절 가능, '잠금' 상태에서는 크기·위치 변경 불가. 도구 모음: 링크·바탕 화면·새 도구 모음
  • 검색: 파일 탐색기 검색(F3, 위치 지정·필터 가능) vs 작업 표시줄 검색(■+S, 컴퓨터 전체와 웹 검색, 필터 불가)

프린터와 스풀

  • 기본 프린터: 한 대만 지정 가능. 네트워크 프린터·공유 프린터도 기본 프린터로 지정 가능
  • 인쇄 대기 중 문서: 삭제·인쇄 순서 변경·일시 중지는 가능, 문서 내용·용지 방향·매수 변경은 불가
  • 스풀(SPOOL): 저속의 프린터와 고속의 CPU를 병행 처리 — 인쇄하는 동안 다른 작업 가능(처리 효율↑, 인쇄 속도 자체가 빨라지는 건 아님)
  • 인쇄 속도 단위: CPS(도트 프린터) · LPM(라인) · PPM(잉크젯/레이저) · IPM(ISO 표준 — 잉크젯 일반 모드 측정) · DPI는 속도가 아니라 해상도

한글 깨짐·언어 설정 7월 제보

  • 유니코드를 지원하지 않는 프로그램에서 한글이 깨질 때: [설정] → [시간 및 언어] → 관련 설정의 국가/지역(관리자 언어 설정 — 유니코드를 사용하지 않는 프로그램용 언어)에서 한국어로 변경
유니코드를 지원하지 않는 프로그램에서 한글이 깨질 때 Windows에서 해결하는 곳은?
정답: [설정]→[시간 및 언어]에서 국가/지역(유니코드를 사용하지 않는 프로그램용 언어) 변경.

제어판·설정 핵심

  • 작업 표시줄: 자동 숨기기, 위치(상하좌우) 변경, 단추 그룹화 설정 가능. 파일 정렬 방식은 작업 표시줄이 아니라 폴더 옵션의 영역
  • 접근성: 돋보기, 내레이터, 화상 키보드, 고대비 등. 다중 디스플레이 설정은 접근성이 아니라 [시스템]-[디스플레이]
  • 디스크 정리: 불필요한 파일 삭제로 공간 확보. 드라이브 조각 모음(최적화): 단편화 해소로 접근 속도 향상(용량이 늘어나는 것이 아님)
  • 작업 관리자: 실행 앱 강제 종료, 프로세스·성능(CPU/메모리) 확인, 시작프로그램 관리. 프로그램 설치/제거는 불가
  • 레지스트리: 시스템·앱 구성 정보 저장 데이터베이스. regedit로 편집, 잘못 수정하면 시스템 오류 위험
"~에서 설정할 수 없는 것은?" 유형이 압도적으로 많습니다. 각 설정 화면(작업 표시줄/접근성/디스플레이/폴더 옵션)의 경계를 묻는 문제 — 기능이 '어느 메뉴 소속인지'를 정확히 구분해 두세요.
한글 Windows 10에서 [설정]→[접근성]에서 구성할 수 없는 항목은?
정답: 다중 디스플레이 구성 — [시스템]→[디스플레이]에서 관리. 돋보기·내레이터·화상 키보드는 접근성 소속.

연결 프로그램·클립보드

  • 연결 프로그램: 파일 확장자별로 "이 파일을 열 앱"을 연결해 두는 것. 더블클릭하면 연결된 앱이 파일을 열어줌. 쉽게 말하면 .txt는 메모장, .hwp는 한글처럼 짝을 정해두는 것
  • 연결 프로그램을 바꿔도 파일 내용은 변하지 않음. 하나의 확장자에 연결 프로그램은 하나(다른 앱으로 바꾸기는 가능)
  • 연결 프로그램이 지정되지 않은 파일을 더블클릭하면 → 앱 선택 대화상자가 나타남(자동 삭제되거나 오류로 종료되지 않음)
  • 클립보드: 복사(Ctrl+C)·잘라내기(Ctrl+X)한 내용이 임시 보관되는 메모리 공간. 모든 앱이 공유하며, 시스템 재시작 시 내용이 사라짐
  • Windows 10은 ■+V(클립보드 기록)로 여러 항목 보관·선택 붙여넣기 가능. 가장 최근 복사본이 기본 붙여넣기 대상

파일 탐색기 — 선택·속성 창

  • 연속 선택 = Shift+클릭, 떨어져 있는 파일 개별 선택 = Ctrl+클릭, 전체 선택 = Ctrl+A
  • 파일·폴더 선택 후 Alt+Enter = [속성] 창 열기
  • 폴더 [속성] 창: 종류·위치·크기·포함된 파일/폴더 수·만든 날짜 확인, 특성(읽기 전용·숨김) 설정, [공유]·[보안]·[이전 버전] 탭 제공. 이름 바꾸기는 속성 창에서 못 함(F2로)
  • 점프 목록: 작업 표시줄 아이콘을 우클릭하면 그 앱에서 최근 사용한 파일·자주 가는 목록이 뜨는 기능. 항목을 목록에 고정할 수도 있음
  • 에어로 피크(Aero Peek): 작업 표시줄 오른쪽 끝(바탕 화면 보기)에 마우스를 올리면 열린 창이 투명해져 바탕 화면을 미리 보고, 클릭하면 모든 창 최소화

[설정] 범주별 경계 — "여기서 못 하는 것" 단골

  • [앱]: 앱 제거·기본 앱 지정·선택적 기능·시작 프로그램. 하드웨어 장치 추가·제거는 [앱]이 아님(장치 관리자)
  • [개인 설정]: 배경, 색, 잠금 화면(화면 보호기 포함), 테마, 글꼴, 시작 메뉴, 작업 표시줄. 해상도 변경은 개인 설정이 아니라 [시스템]→[디스플레이]
  • [시스템]→[정보]: Windows 사양·장치 사양 확인, PC 이름 바꾸기. / [시스템]→[소리]: 출력·입력 장치 선택, 볼륨(마스터·앱별) 조정
  • 사용자 계정: 관리자 계정 vs 표준 계정. 계정 유형 변경·이름 변경 가능. ■+L로 로그아웃 없이 사용자 전환 — 앞 사용자의 작업은 유지됨
  • 키보드 속성: 재입력 시간·반복 속도·커서 깜박임 속도 조정(키보드 종류 자체를 바꾸는 곳 아님). 마우스 설정: 기본 단추 왼/오 전환, 커서 속도, 휠 스크롤 줄 수

드라이브·저장 공간 관리

  • NTFS: FAT32보다 보안(권한)·안정성·대용량 지원이 우수한 Windows 기본 파일 시스템. 압축·암호화 지원
  • [포맷] 대화상자에서 지정: 용량·파일 시스템·할당 단위 크기·볼륨 레이블·빠른 포맷. 파티션 나누기·불량 섹터 복구는 못 함
  • 디스크 공간 부족 해결: 디스크 정리, 사용 안 하는 앱 제거, 휴지통 비우기, 파일 압축. 조각 모음은 속도 개선이지 공간 확보가 아님 — 단골 함정
  • 백업(파일 히스토리): 원본 손상에 대비해 다른 드라이브에 복사본을 주기적으로 저장. 백업은 별도 드라이브에 하는 것이 원칙

명령 프롬프트·기본 앱·장치 관리

  • ■+R(실행) → cmd 입력 = 명령 프롬프트 창. 텍스트 명령으로 시스템을 다루는 창(그래픽 설정 화면이 아님)
  • PING: 상대 컴퓨터까지 네트워크가 연결되는지 응답 시간으로 점검하는 명령
  • 메모장: 서식 없는 텍스트(.txt) 전용 — 그림·표 삽입 불가, OLE 개체 연결 불가. F5 = 시간/날짜 삽입, 첫 줄에 .LOG 입력 시 열 때마다 시간 자동 기록
  • 유틸리티: 압축·백신·디스크 관리처럼 컴퓨터 사용을 돕는 보조 프로그램. 압축의 목적은 공간 절약·전송 시간 단축(보안 강화가 아님)
  • 장치 관리자: 하드웨어가 정상 동작하는지 확인, 드라이버 업데이트·제거. PnP(플러그 앤 플레이): 장치를 연결하면 자동 감지 + 드라이버 자동 설치까지 해주는 규격
  • 프린터 설치: [설정]→[장치]→[프린터 및 스캐너]에서 추가. 네트워크 프린터도 로컬처럼 설치·기본 지정 가능. 레이저 프린터 = 토너·드럼 방식, 인쇄 속도 빠르고 소음 적음
쉽게 말하면 [설정] 문제는 "이 메뉴 소속이 맞나?"를 묻는 자리 찾기 게임입니다. 화면 보호기=잠금 화면(개인 설정), 해상도=디스플레이, PC 이름=정보, 시작 프로그램=앱·작업 관리자 — 소속만 정확히 외우면 점수가 됩니다.

대표 기출 18제 — 풀고 넘어가기

문제 1 2021년 기출
다음 중 파일을 삭제했을 때 [휴지통]에 임시로 저장되어 원래대로 되돌릴 수 있는 상황은?
  1. ① 바탕 화면에 있는 파일을 [휴지통]으로 드래그 앤 드롭하여 삭제한 경우
  2. ② USB 메모리에 저장되어 있는 파일을 [Delete]로 삭제한 경우
  3. ③ 네트워크 드라이브의 파일을 바로 가기 메뉴의 [삭제]를 클릭 하여 삭제한 경우
  4. ④ [휴지통]의 크기를 0%로 설정한 후 파일 탐색기 안의 파일을 삭제한 경우
문제 2 2022년 기출
다음 중 레지스트리(Registry)에 대한 설명 중 옳지 않은 것은?
  1. ① 레지스트리를 잘못 편집하면 운영체제를 완전하게 다시 설치해야 하는 심각한 문제가 발생할 수 있으나 데이터의 손실은 방지해 준다.
  2. ② Windows 운영체제에서 사용되는 설정값과 여러 시스템 관련 정보가 보관되어 있는 계층적 구조의 데이터베이스이다.
  3. ③ [시작] 단추에서 마우스 우클릭을 한 후 [실행]을 선택하고, 나타나는 창에 regedit를 입력한 다음 확인을 눌러서 열 수 있다.
  4. ④ 작업 표시줄에 있는 검색 입력란에 regedit를 검색하면 결과 목록에서 레지스트리 편집기를 선택하여 실행할 수 있다.
문제 3 2022년 기출
한글 Windows 10 운영체제에서 하드 디스크에 보관된 파일들을 재정렬하는 단편화 해소 절차를 실행하여 디스크의 파일 접근 및 저장 속도를 개선하는 유틸리티는 무엇인가?
  1. ① 디스크 검사
  2. ② 디스크 정리
  3. ③ 디스크 포맷
  4. ④ 드라이브 조각 모음 및 최적화
문제 4 2022년 기출
한글 Windows 10 환경에서 하드디스크를 포맷할 때 나타나는 [포맷] 대화상자에서 조정 불가능한 항목을 고르세요.
  1. ① 볼륨 레이블 입력
  2. ② 파티션 제거
  3. ③ 파일 시스템 선택
  4. ④ 빠른 포맷 선택
문제 5 2022년 기출
다음 중 한글 Windows 10의 인쇄 기능에 관한 설명 중 옳지 않은 것은?
  1. ① 기본 프린터는 인쇄할 때 별도로 프린터를 선택하지 않으면 자동으로 지정되어 사용되는 프린터를 의미한다.
  2. ② 프린터 속성 대화상자에서는 용지 종류, 공유 여부, 포트 설정 등 다양한 항목을 조정할 수 있다.

문제 1 — 정답 ①
휴지통은 내부 하드디스크에서 표준 방법으로 삭제할 때만 거칩니다. USB 등 이동식 매체·네트워크 드라이브에서 삭제하거나 휴지통 용량이 0%면 즉시 삭제되어 복구할 수 없습니다.
문제 2 — 정답 ①

문제 3 — 정답 ④

문제 4 — 정답 ②
[포맷] 대화상자에서는 볼륨 레이블·파일 시스템·빠른 포맷을 지정할 수 있지만, 파티션 제거는 '디스크 관리'에서 하는 별도 작업입니다.
문제 5 — 정답 ④
인쇄 중인 작업도 프린터 큐에서 선택해 취소(삭제)할 수 있습니다. '취소 불가능'이 오류입니다.
문제 6 — 정답 ①
연결 프로그램을 제거해도 데이터 파일 자체는 삭제되지 않습니다. 연결 프로그램은 파일을 여는 앱일 뿐 파일과 별개입니다.
