📌오늘 배운 핵심 요약

필기

리본 메뉴: 탭 → 그룹 → 명령

엑셀 2007부터 도입된 리본 메뉴는 탭, 그룹, 명령 순서로 구성됩니다. 아이콘 위에 마우스를 올리면 스크린 팁이 설명을 보여줘요.

예: 데이터 탭 → 데이터 도구 그룹 → 데이터 유효성 검사 명령
실기

수식 표시 Ctrl+`

화면엔 결과값(7)이 보여도 실제로는 수식(=5+2)이 들어있을 수 있어요. Ctrl+`로 전체 수식을 한눈에 확인합니다.

실무: 함수 오류를 찾을 때 셀을 하나씩 안 눌러도 됨
필기

메모는 Delete로 안 지워진다

Delete는 셀 내용만 지웁니다. 메모까지 지우려면 '모두 지우기' 또는 '메모 삭제'를 써야 해요. 메모 삽입은 검토 탭(Shift+F2)입니다.

시험 단골: Delete / 내용 지우기 / 모두 지우기의 차이
필기

채우기 핸들 + Ctrl은 반대 동작

숫자 하나는 복사가 기본, Ctrl 누르면 1씩 증가. 숫자 두 개는 패턴 증가가 기본, Ctrl 누르면 반복 복사됩니다.

시험 단골: "1, 3" 드래그 → 1,3,5,7 / Ctrl 드래그 → 1,3,1,3
실기

데이터 유효성 검사

셀에 입력할 수 있는 값을 제한하는 기능. 범위를 먼저 선택하고 데이터 탭에서 실행합니다. 목록·정수 등으로 제한할 수 있어요.

실무: 잘못된 데이터에 빨간 동그라미 표시로 오타 찾기
필기

와일드카드 * 와 ?

찾기(Ctrl+F)/바꾸기(Ctrl+H)에서 *는 글자 수 제한 없이, ?는 딱 한 글자만 대신합니다.

예: 김* = 김으로 시작하는 전부, 김?? = 김으로 시작하는 세 글자

(필기) 메모(노트) — 포스트잇으로 이해하기

어디에 있나 새 내용을 넣으니 삽입 탭일 것 같지만, 문서를 검토하며 의견을 남기는 기능이라 검토 탭에 있어요. 단축키는 Shift+F2.
삭제 규칙 Delete·내용 지우기는 셀 내용만 지우고 메모는 남습니다. 메모까지 지우려면 '모두 지우기' 또는 '메모 삭제'를 사용하세요.
자동 크기 메모 테두리를 클릭해 메모 자체를 선택한 뒤 → 마우스 오른쪽 → 메모 서식 → 맞춤 탭 → 자동 크기. 글자 안을 클릭한 상태면 탭이 하나만 나와요.
인쇄 기본은 인쇄 안 됨. 페이지 설정 → 시트 탭 → 메모에서 '시트에 표시된 대로' 또는 '시트 끝'을 선택하면 인쇄돼요.
시험 예시: "셀 내용을 Delete로 지우면 메모도 삭제된다" → 틀린 설명. 메모가 삽입된 셀을 이동하면 메모도 함께 이동하지만, 채우기 핸들 복사로는 메모가 복사되지 않습니다.

(필기) 채우기 핸들 — Ctrl은 항상 "원래 기능의 반대"

숫자 1개 그냥 드래그 = 복사(1,1,1) / Ctrl 드래그 = 1씩 증가(1,2,3).
숫자 2개 두 셀을 선택하고 드래그하면 차이만큼 증가(1,3 → 1,3,5,7) / Ctrl 드래그 = 반복 복사(1,3,1,3).
문자+숫자 혼합 가장 오른쪽 숫자가 1씩 증가(서울1 → 서울2, 서울3). "컴활2급"처럼 증가하면 안 될 땐 Ctrl로 복사.
날짜 기본은 하루씩 증가. 채우기 옵션은 일/평일/월/연 단위 — 주 단위는 없어요.
사용자 지정 목록 월→화→수, 1분기→2분기처럼 등록된 목록은 순서대로 채워집니다. 파일 → 옵션 → 고급 → 사용자 지정 목록 편집에서 직접 추가 가능(예: 사원→대리→과장).
시험 예시: 빈 시트에 1부터 10,000까지 넣으라면 채우기 핸들 대신 홈 탭 → 채우기 → 계열(방향: 열, 유형: 선형, 단계값 1, 종료값 10000)을 사용합니다.

(실기) 데이터 유효성 검사 — 잘못된 입력을 미리 막는 문지기

순서 ① 범위를 먼저 선택 → ② 데이터 탭 → 데이터 도구 그룹 → 데이터 유효성 검사 → ③ 제한 대상 설정.
목록 제한 HSD1,HSD2,HSD3처럼 정해진 값만 입력하게 하려면 '목록'. 직접 입력 시 항목 사이는 쉼표로 구분, 셀 범위 지정도 가능.
정수 제한 단가를 16000~20000 사이로 제한하는 식으로 숫자 범위를 지정할 수 있어요.
이미 입력된 데이터 '잘못된 데이터' 기능을 쓰면 조건에 안 맞는 값에 빨간 동그라미가 표시됩니다. 값을 고치면 표시가 사라져요.
탭 구성 설정(조건) / 설명 메시지(셀 선택 시 안내) / 오류 메시지(잘못 입력 시 경고) / IME 모드(한글·영문 입력 상태).
시험 예시: "유효성 검사는 입력 전에만 설정할 수 있다" → 틀린 설명. 입력 후에도 설정해서 잘못된 데이터를 찾아낼 수 있습니다.

(필기) 찾기/바꾸기와 와일드카드 — *는 자유, ?는 한 글자

단축키 찾기는 Ctrl+F(Find), 바꾸기는 Ctrl+H. 찾기 범위는 시트 또는 통합 문서, 찾는 위치는 수식 또는 값.
별표 * 글자 수 제한 없이 모든 문자를 대신해요. 김* = 김으로 시작하는 전부, *주식회사 = 주식회사로 끝나는 전부.
물음표 ? 딱 한 글자만 대신해요. 김?? = 김으로 시작하는 세 글자 이름, 삼?주식회사 = 여섯 글자.
전체 셀 내용 일치 체크하면 셀 내용 전체가 조건과 정확히 맞아야 검색되고, 해제하면 일부만 포함돼도 검색됩니다.
시험 예시: 셀에 =56+7이 있고 결과가 63일 때 — '값'에서 찾으면 63으로, '수식'에서 찾으면 56+7 내용으로 검색됩니다.

(실기) 이동·선택·입력 보조 — Ctrl은 점프, Shift는 선택

점프 Ctrl+방향키 = 데이터 끝으로 이동, Ctrl+Home = A1, Ctrl+End = 마지막 데이터 셀, Ctrl+Page Up/Down = 시트 이동.
선택 Shift+방향키 = 한 칸씩 선택, Ctrl+Shift+방향키 = 데이터 끝까지 한 번에 선택. "점프해서 선택"으로 외우세요.
자동 완성 같은 열의 문자 데이터는 앞 글자만 입력해도 자동 완성돼요. 숫자·날짜·시간은 안 됩니다. Alt+↓로 기존 입력값 목록을 볼 수 있어요.
한자·특수문자 한글 입력 후 한자 키로 변환, 특수문자는 자음(ㅁ 등) 입력 후 한자 키 → 번호나 Page Up/Down으로 선택.
수정·삭제 F2 = 셀 안에 커서 넣어 부분 수정. 범위 선택 후 Delete = 전체 삭제, Backspace = 첫 번째 셀만 삭제.
시험 예시: 데이터 범위 안에서 Ctrl+A 한 번 = 현재 데이터 범위 선택, 두 번 = 시트 전체 선택.

⭐ 가장 많이 틀린 부분

메모를 지우려고 Delete를 눌러요

Delete와 내용 지우기는 셀 내용만 지우고 메모는 그대로 남아요. 시험에서 가장 자주 나오는 함정입니다.

메모 삭제는 검토 탭 → 삭제, 마우스 오른쪽 → 메모 삭제, 또는 홈 탭 → 지우기 → 메모 지우기/모두 지우기를 사용하세요.

자주 하는 실수

채우기 핸들의 Ctrl 기능을 반대로 기억해요

Ctrl은 원래 동작의 반대입니다. 숫자 하나는 원래 복사 → Ctrl 누르면 증가, 숫자 두 개는 원래 패턴 증가 → Ctrl 누르면 반복 복사예요.

"Ctrl = 반대"로 외우면 어떤 조합이 나와도 풀 수 있어요.

자주 하는 실수

날짜 채우기에 '주 단위'가 있다고 생각해요

날짜 자동 채우기 옵션은 일 단위, 평일 단위, 월 단위, 연 단위 4가지뿐이에요. 시험 보기에 '주 단위 채우기'가 나오면 함정입니다.

"일·평일·월·연 — 주는 없다"로 기억하세요.

자주 하는 실수

메모 서식을 열었는데 탭이 하나만 나와요

메모 안에 커서가 들어간 상태(글자 편집 중)에서 서식을 열면 글꼴 탭만 나와요. 자동 크기 설정을 못 찾는 이유가 대부분 이거예요.

메모 테두리에 마우스를 올려 상하좌우 화살표가 나올 때 클릭 → 메모 자체가 선택된 상태에서 메모 서식을 열어야 맞춤 탭이 보여요.

자주 하는 실수

와일드카드 *와 ?를 헷갈려요

*는 글자 수 제한 없이 여러 글자를, ?는 딱 한 글자만 대신합니다.

"별(*)은 크니까 많이, 물음표(?)는 하나만" — 김??는 세 글자 이름만 찾아요.
