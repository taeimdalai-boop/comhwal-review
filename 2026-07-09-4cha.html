📌오늘 배운 핵심 요약

시트 보호 vs 통합 문서 보호

시트 보호는 셀 편집(입력·수정)을 제한하고, 통합 문서 보호는 시트 삽입·삭제·이동·숨기기·이름 바꾸기 등 시트 자체의 구조 변경을 제한합니다. 자주 반대로 헷갈립니다.

암기: "시트 보호=내용, 통합 문서 보호=구조"

문서 암호는 저장할 때 건다

파일 → 다른 이름으로 저장 → 도구 → 일반 옵션에서 열기 암호, 쓰기 암호, 읽기 전용 권장, 백업 파일 항상 만들기를 지정할 수 있습니다.

열기 암호=아예 못 엶 / 쓰기 암호=열람은 되지만 수정 저장 못 함

표시 형식 코드 복습: # vs 0

# 은 의미 없는 0을 표시하지 않고, 0 은 값이 0이어도 그대로 표시합니다. #,##0"원"처럼 조합해 천단위 콤마와 "0원" 표시를 동시에 만듭니다.

시험 단골: 0원일 때 0을 표시하라는 조건이면 무조건 0을 넣은 코드

파일 형식 구분: xlsx / xlsm / xlsb / xls / csv

xlsx는 일반 통합 문서, xlsm은 매크로 포함, xlsb는 용량을 줄이는 바이너리 형식, xls는 구버전 호환용, csv는 쉼표로 구분된 순수 텍스트 형식입니다.

매크로가 들어간 파일은 반드시 xlsm으로 저장

인쇄 영역은 미리 보기가 아니라 페이지 레이아웃에서

인쇄할 범위를 선택한 뒤 페이지 레이아웃 → 인쇄 영역 → 인쇄 영역 설정으로 지정합니다. 인쇄 미리 보기 화면에서는 반복할 행/열이나 인쇄 영역을 새로 설정할 수 없습니다.

미리 보기에서 손봐야 할 게 있으면 Esc로 나온 뒤 설정

페이지 나누기: 점선(자동) vs 실선(수동)

페이지 나누기 미리 보기 화면에서 점선은 엑셀이 자동으로 잡은 경계, 실선은 사용자가 직접 드래그해 만든 수동 경계입니다. 회색 영역은 삭제된 게 아니라 인쇄에서만 제외된 것입니다.

균등 배치: 전체 레코드 수 ÷ 목표 페이지 수 → 나누기 선 위치 조정

정렬은 최대 64개 기준, 병합 셀은 금지

정렬 기준은 최대 64개까지 추가할 수 있고, 셀 값·셀 색·글꼴 색을 기준으로도 정렬할 수 있습니다. 단, 정렬 범위에 병합된 셀이 있으면 정렬 자체가 실행되지 않습니다.

오름차순 순서: 숫자→특수문자→영문 소문자→영문 대문자→한글→FALSE→TRUE→오류값, 빈 셀은 항상 맨 뒤

시트 보호 & 통합 문서 보호 — 서식 파일(폼) 만들기

기본 속성 모든 셀은 기본적으로 잠금(Lock) 속성이 체크돼 있습니다 (Ctrl+1 → 보호 탭).
폼 만들기 순서 ①입력받을 흰 셀들만 Ctrl로 선택 → ②Ctrl+1 → 보호 탭 → 잠금 체크 해제 → ③검토 탭 → 시트 보호 실행.
잠긴 셀 선택 옵션 이 옵션을 해제하면 잠긴 셀로는 마우스 클릭도, 셀 포인터 이동도 안 되는 완전한 입력 폼이 됩니다.
통합 문서 보호 셀 내용이 아니라 시트 삽입·삭제·이동·숨기기·이름 바꾸기 같은 시트 구조 변경을 막습니다. 실무에서는 시트 보호가 더 많이 쓰입니다.
시험 예시: "통합 문서 보호를 하면 셀 입력이 안 된다"는 틀린 설명 — 그건 시트 보호의 기능입니다.

문서 암호 설정

경로 파일 → 다른 이름으로 저장 → 찾아보기 → 도구(하단) → 일반 옵션.
열기 암호 입력하지 않으면 파일 자체를 열 수 없습니다.
쓰기 암호 열람은 가능하지만, 암호를 모르면 수정 후 저장(덮어쓰기)이 불가능합니다.
기타 옵션 읽기 전용 권장(열 때 경고만 표시), 백업 파일 항상 만들기(이전 버전을 자동 보관).
시험 예시: 열기 암호와 쓰기 암호는 같은 대화상자의 서로 다른 입력란입니다. 둘 다 설정할 수도 있습니다.

표시 형식 복습 — 겉모습과 실제 값은 다르다

숫자 # 은 불필요한 0을 표시하지 않고, 0 은 자릿수를 강제로 채웁니다. #,###"원"과 #,##0"원"의 차이는 값이 0일 때 "0"을 표시하느냐입니다.
문자 @ 는 셀에 입력된 문자를 그대로 가져옵니다. @"지점"으로 설정하면 '서울' 입력 시 '서울지점'으로 표시됩니다.
날짜·시간 yy/yyyy(연도), mmm/mmmm(월 영문), ddd/dddd(요일 영문), aaa/aaaa(요일 한글). h/m/s로 시·분·초를 표시합니다.
핵심 원리 숫자 뒤에 문자를 직접 입력하면 그 순간 문자 데이터가 되어 계산이 불가능해집니다. 반드시 표시 형식으로 겉모습만 바꿔야 합니다.
시험 예시: 화면에 '서울 지점'이라 보여도 실제 값은 '서울'일 수 있으므로, 함수 조건식에는 수식 입력줄의 원본 값을 기준으로 판단해야 합니다.

파일 형식 5종

xlsx 일반적인 Excel 통합 문서 형식.
xlsm 매크로(VBA)가 포함된 통합 문서. 매크로 유지하려면 반드시 이 형식으로 저장.
xlsb 이진(바이너리) 형식으로 저장해 파일 용량을 줄일 때 사용.
xls 구버전 Excel과 호환되는 예전 형식.
csv 데이터를 쉼표(,)로 구분한 순수 텍스트 형식. 서식은 저장되지 않습니다.
시험 예시: 매크로가 포함된 파일을 xlsx로 저장하면 매크로가 제거된다는 경고가 뜹니다 — xlsm으로 저장해야 합니다.

인쇄 영역 · 페이지 설정

인쇄 영역 지정 인쇄할 범위를 선택 → 페이지 레이아웃 → 인쇄 영역 → 인쇄 영역 설정.
페이지 설정 4개 탭 페이지(용지 방향·배율·자동 맞춤·시작 페이지 번호), 여백(상하좌우 여백·가로/세로 가운데 맞춤), 머리글/바닥글(왼쪽·가운데·오른쪽 구역에 날짜·페이지 번호·파일명 삽입), 시트(인쇄 영역·반복할 행/열·눈금선·행/열 머리글·메모·간단하게 인쇄).
반복할 행 여러 페이지로 나뉘는 표에서 제목(필드명) 행을 모든 페이지 위쪽에 자동 반복시킵니다. 페이지 설정 → 시트 → 반복할 행에서 지정.
주의 프린터 선택은 페이지 설정이 아니라 인쇄 화면(Ctrl+P)에서 합니다. 인쇄 미리 보기 화면에서는 반복할 행/열이나 인쇄 영역을 새로 설정할 수 없습니다.
시험 예시: 인쇄할 표 아래에 열을 하나 더 끼워 넣어 가운데처럼 보이게 하는 건 잘못된 방법 — 여백 탭의 '페이지 가운데 맞춤'을 써야 합니다.

페이지 나누기 미리 보기 — 자동(점선) vs 수동(실선)

진입 보기 탭 → 페이지 나누기 미리 보기.
자동 페이지 나누기 엑셀이 용지 크기에 맞춰 자동으로 잡은 경계선으로, 점선으로 표시됩니다.
수동 페이지 나누기 사용자가 행 머리글을 우클릭해 페이지 나누기를 삽입하거나, 자동 경계선을 드래그해 위치를 옮기면 실선으로 바뀝니다.
회색 영역 데이터가 삭제된 게 아니라 인쇄 대상에서 제외된 영역입니다. 흰색 영역만 인쇄됩니다.
레코드 균등 배치 레코드는 데이터 목록의 각 행(필드명 제외)을 의미합니다. 전체 레코드 수를 목표 페이지 수로 나눈 뒤, 페이지 나누기 선을 드래그해 각 페이지의 레코드 수를 맞춥니다.
주의 행 높이나 열 너비를 바꾸면 자동 페이지 나누기 위치도 함께 달라집니다.
시험 예시: 224개의 레코드를 7페이지에 균등 배치하면 한 페이지당 32개입니다 (필드명 행은 레코드 수에서 제외).

정렬 — 오름차순 규칙과 다중 기준

기본 정렬 데이터 탭 → 정렬 및 필터 → 오름차순/내림차순. 셀 포인터가 있는 필드를 기준으로 정렬됩니다.
오름차순 순서 숫자 → 특수문자 → 영문 소문자 → 영문 대문자 → 한글 → 논리값(FALSE→TRUE) → 오류값. 빈 셀은 오름차순·내림차순 모두 항상 맨 마지막에 배치됩니다.
다중 기준 정렬 데이터 탭 → 정렬 대화상자에서 기준을 최대 64개까지 추가할 수 있습니다. 셀 값뿐 아니라 셀 색, 글꼴 색으로도 정렬 가능합니다.
사용자 지정 목록 정렬 정렬 기준이 '셀 값'일 때만 사용할 수 있고, 셀 색·글꼴 색 정렬에는 적용되지 않습니다.
정렬 방향 정렬 → 옵션 → '왼쪽에서 오른쪽'을 선택하면 행이 아니라 열 방향으로 정렬됩니다.
⚠️ 병합 셀 주의 정렬 범위에 병합된 셀이 포함되면 정렬 자체가 실행되지 않습니다. 정렬 전 병합을 해제해야 합니다.
시험 예시: 등수(오름차순)와 점수(내림차순)처럼 항목에 따라 정렬 방향을 다르게 판단해야 합니다.

시트 보호와 통합 문서 보호를 반대로 알고 있어요

"통합 문서 보호를 하면 셀 편집이 안 된다"고 착각하는 경우가 많습니다. 셀 편집을 막는 건 시트 보호이고, 통합 문서 보호는 시트 삽입·삭제·이동·이름 변경 같은 구조 변경만 막습니다.

"시트 보호=내용(셀), 통합 문서 보호=구조(시트)"로 짝지어 외우세요.

자주 하는 실수

회색 영역의 데이터가 지워진 줄 알아요

페이지 나누기 미리 보기에서 회색으로 표시된 영역은 인쇄 대상에서 제외됐을 뿐, 데이터는 그대로 남아 있습니다.

파란색 경계선을 드래그해서 인쇄 영역(흰색 부분)을 다시 조정하면 됩니다.

자주 하는 실수

인쇄 미리 보기 화면에서 반복할 행을 설정하려고 해요

인쇄 미리 보기 화면에서는 반복할 행/열, 인쇄 영역 등을 새로 설정할 수 없습니다. Esc로 빠져나온 뒤 페이지 레이아웃 탭에서 설정해야 합니다.

"미리 보기는 확인만, 설정 변경은 페이지 레이아웃 탭에서"로 기억하세요.

자주 하는 실수

숫자 뒤에 '원'을 직접 입력해요

화면엔 "10000원"처럼 보여도, 직접 문자를 입력하면 더 이상 숫자가 아니라 문자 데이터가 되어 함수 계산이 불가능해집니다.

값은 숫자 그대로 두고, 표시 형식에 #,##0"원"을 지정하세요.

자주 하는 실수

병합된 셀이 포함된 채로 정렬을 시도해요

정렬 범위 안에 병합 셀이 하나라도 있으면 정렬 명령 자체가 실행되지 않고 오류 메시지가 뜹니다.

정렬 전에는 반드시 병합을 모두 해제하세요.
