특수문자 한글 자음(ㄱ, ㄴ, ㄷ 등)을 입력한 뒤 한자 키를 누르면 특수문자 목록이 뜹니다. 가장 많이 쓰이는 자음은 미음(ㅁ)이에요.
목록 이동 마우스 대신 Page Up / Page Down 키로 이동하면 빠르고, 선택할 때는 숫자 키패드를 쓰면 더 편리합니다.
시험 예시: "특수문자를 입력하려면 반드시 마우스로 목록을 클릭해야 한다" → 틀린 설명. 키보드만으로도 입력 가능합니다.

(필기) 메모 vs 윗주 — 반드시 구분해서 외우기

메모(노트) Shift+F2로 삽입. Delete로 셀 내용을 지워도 메모는 남습니다. 메모 테두리 선택(상하좌우 화살표) 후 우클릭 → 메모 서식 → 맞춤 탭 → 자동 크기로 크기를 조절해요.
윗주(위쪽에 작은 글자) 문자 데이터에만 입력 가능 — 숫자(예: 604)에는 설정해도 화면에 표시되지 않습니다.
윗주 서식 일부분만 골라 서식을 바꿀 수 없고, 전체 글꼴만 한 번에 바꿀 수 있어요.
윗주 삭제 규칙 셀 안의 본문 텍스트를 지우면 위에 달린 윗주도 함께 자동으로 삭제됩니다. 이것이 메모와 가장 큰 차이점이에요.
표시 방법 홈 탭 → 글꼴 그룹의 윗주 필드 표시/숨기기 버튼을 눌러 켜고 끕니다.
시험 예시: "메모와 윗주 모두 본문을 지우면 함께 사라진다" → 틀린 설명. 메모는 안 지워지고, 윗주만 함께 지워집니다.

(필기) 채우기 핸들 — 순수 데이터, 혼합 데이터, 사용자 지정 목록

순수 데이터 숫자만 또는 문자만 있으면 드래그 시 기본적으로 복사됩니다.
숫자 연속 데이터 채우기 4가지 방법 ①드래그 후 자동 채우기 옵션에서 '연속 데이터 채우기' 선택 ②Ctrl 누른 채 드래그 ③패턴을 보여주는 두 셀(1, 2)을 함께 잡고 드래그 ④홈 탭 → 편집 → 채우기 → 계열(방향: 열, 종료값 입력).
혼합 데이터·날짜 문자+숫자 조합은 마지막 숫자가 1씩 증가하고, 날짜는 하루씩 증가합니다. 그대로 복사하고 싶으면 Ctrl을 누른 채 드래그하세요.
사용자 지정 목록 '갑을병정', '자축인묘', '월화수목금토일'은 내장돼 있어 자동 채우기가 되지만, '가나다라', 'ABCD'는 기본 목록에 없어 드래그해도 복사만 됩니다.
시험 예시: "가나다라"를 드래그하면 "마바사아"로 이어진다 → 틀린 설명. 기본 목록에 없어서 그대로 복사됩니다.

(실기) 데이터 유효성 검사 & 자동 고침 옵션

순서 범위를 먼저 지정한 뒤 데이터 탭 → 데이터 유효성 검사를 실행합니다.
목록 제한 제한 대상을 '목록'으로 하고, 원본에 쉼표(,)로 구분해 입력합니다 (예: 월,화,수,목,금,토,일).
숫자 제한 제한 대상을 '정수'로 하고 최소값(0)·최대값(100)을 지정합니다.
자동 고침 옵션 파일 → 옵션 → 언어 교정 → 자동 고침 옵션에서 줄임말(우컴) → 정식 명칭(우리컴퓨터학원)을 등록하면 입력 시 자동 변환됩니다.
기본 서식 예시 (c) 입력 시 ©로, (tel) 입력 시 전화기 모양 기호로 자동 변경됩니다.
시험 예시: 목록 제한의 원본 항목 구분자는 세미콜론(;)이 아니라 쉼표(,)입니다.

(필기) 행/열 너비 조절 및 숨기기·숨기기 취소

자동 조절 열 머리글 경계선을 더블 클릭하면 가장 긴 데이터에 맞춰 너비가 자동 조절됩니다.
전체 동일 조절 시트 전체 선택 후 열 머리글 경계선을 드래그하면 모든 열이 같은 너비로 바뀝니다.
숨기기 열/행 머리글 선택 후 우클릭 → 숨기기. 숨겨진 부분은 머리글 경계선이 이중 실선으로 표시됩니다.
숨기기 취소 4가지 방법 ①숨겨진 열의 좌우 머리글을 포함해 범위 지정 후 우클릭 → 숨기기 취소 ②이중 실선에 마우스를 대고 더블 클릭 ③시트 전체 선택 후 열 너비를 특정 숫자로 일괄 변경 ④이름 상자에 A1 입력 후 엔터 → 홈 탭 → 서식 → 숨기기 및 숨기기 취소 → 열/행 숨기기 취소.
⚠️ 주의(필기 기출) 숨겨진 행/열은 인쇄되지 않으며, 정렬(Sort) 시 위치가 이동하지 않으므로 정렬 전 반드시 숨기기를 해제해야 합니다.
시험 예시: A1 셀이 숨겨져 안 보일 때는 이름 상자에 직접 'A1'을 입력해 찾아간 뒤 숨기기 취소를 실행합니다.

(필기) 워크시트 관리 및 그룹화

시트 선택 연속 선택은 Shift, 비연속 선택은 Ctrl을 누른 채 클릭합니다.
시트 추가 단축키는 Shift+F11이며, 현재 시트의 왼쪽(앞)에 추가됩니다. 우측 하단의 + 버튼은 반대로 오른쪽(뒤)에 추가돼요.
시트 삭제 삭제된 시트는 Ctrl+Z(실행 취소)로 되살릴 수 없는 영구 삭제입니다.
그룹화 특징 제목 표시줄에 [그룹]이 표시되고, 삽입 탭(도형·차트 등)과 데이터 탭(정렬·필터·유효성검사)의 대다수 기능이 비활성화됩니다.
그룹 활용 여러 시트를 묶어 데이터를 입력하면 모든 시트의 동일한 셀 위치에 동시 입력되어 1월~12월 장부 틀을 짤 때 유용합니다. Ctrl 누른 채 드래그하면 여러 시트를 한 번에 복사할 수 있어요.
그룹 해제 그룹에 포함되지 않은 다른 시트를 클릭하면 자동으로 풀립니다.
시험 예시: 그룹 상태에서는 차트를 삽입할 수 없다 → 맞는 설명. 삽입 탭의 그래픽 개체 기능은 비활성화됩니다.

(실기) 선택하여 붙여넣기 — 결제란 만들기

목적 표의 행/열 너비와 상관없이 깔끔한 결제란을 배치하기 위해 그림으로 붙여넣기를 활용합니다.
텍스트 세로 쓰기 홈 탭 → 맞춤 그룹의 방향(가나 화살표 모양) → '세로 쓰기'를 선택합니다.
그림으로 붙여넣기 원본 복사(Ctrl+C) 후 우클릭 → 선택하여 붙여넣기에서 선택. 원본과 링크가 끊긴 순수 이미지로 고정되어, 원본 시트를 나중에 지워도 안전합니다.
연결된 그림으로 붙여넣기 원본 셀 텍스트를 수정(예: 담당 → 사원)하면 붙여넣은 그림 속 텍스트도 실시간 자동 업데이트됩니다. 단, 원본 시트를 삭제하면 그림이 깨집니다.
시험 예시: 원본이 삭제돼도 안전한 것은 '그림으로 붙여넣기'이고, 원본과 실시간 연동되는 것은 '연결된 그림으로 붙여넣기'입니다.

(실기) 시트 보호 및 통합 문서 보호

기본 속성 모든 셀은 기본적으로 '잠금' 속성이 체크돼 있습니다 (Ctrl+1 → 보호 탭에서 확인 가능).
시트 보호 검토 탭에서 실행하며, 활성화하는 순간 기본적으로 잠겨 있던 모든 셀의 편집이 막힙니다.
서식 파일(폼) 만들기 순서 ①사용자가 입력해야 할 흰색 셀들만 Ctrl로 지정 → ②Ctrl+1(셀 서식) → 보호 탭 → '잠금' 체크 해제 → ③검토 탭 → 시트 보호 실행.
잠긴 셀 선택 옵션 이 옵션을 체크 해제하면 사용자는 마우스로 잠긴 셀을 클릭조차 할 수 없고, 입력 가능한 셀로만 포인터가 이동하는 완벽한 폼 문서가 완성됩니다.
통합 문서 보호 시트 안의 내용이 아니라 시트 자체의 구조(삽입, 삭제, 이동, 이름 바꾸기 등)를 못 하도록 막는 기능입니다.
시험 예시: 시트 보호는 셀 내용을, 통합 문서 보호는 시트 구조 자체를 지키는 기능으로 구분해서 외웁니다.

(필기) 표시 형식 — 겉모습과 실제 값은 다르다

기본 원리 문자 뒤에 직접 '원'을 타이핑하면 숫자가 아닌 문자로 인식되어 함수 계산이 불가능해집니다. 표시 형식으로 겉모습만 바꾸고 실제 값은 숫자로 유지해야 해요.
숫자 서식 코드 # 은 의미 없는 0을 표시하지 않고, 0 은 자릿수를 강제로 채우며 값이 0일 때도 '0'을 표시합니다.
★시험 단골 #,##0 은 세 자리마다 콤마를 찍고, 값이 0원일 때 '0원'으로 명확히 표시하는 서식입니다. 작성 예시: #,##0"원"
문자 서식 코드 @ 는 입력된 문자 데이터 자체를 의미합니다. 사용자 지정에 @"지점"을 설정하면 '서울' 입력 시 화면에 '서울지점'으로 표시됩니다.
날짜·시간 서식 코드 yy/yyyy(연도 2자리/4자리), m~mmmm(월, 개수에 따라 숫자 또는 영문 텍스트), d/dd(일), aaa(한글 요일 1글자, 예: 금)/aaaa(한글 요일 3글자, 예: 금요일).
시험 예시: 셀 서식을 #,##0"원"으로 지정하면 0이 입력된 셀은 "0원"으로 표시됩니다.

(필기) 틀 고정 및 창 나누기 — 화면 보기 전용 기능

공통점 인쇄에는 영향을 주지 않고, 오직 작업 화면 보기만 제어합니다 (보기 탭 → 창 그룹).
틀 고정 핵심 규칙 항상 스크롤되기 시작할 첫 번째 셀을 선택해야 하며, 틀 고정 선은 항상 셀 포인터의 '왼쪽'과 '위쪽'에 생깁니다. 예: 1~3행과 A~B열을 고정하려면 C4 셀을 클릭한 상태에서 틀 고정을 실행합니다.
창 나누기 서로 멀리 떨어진 데이터(예: 4행과 100행)를 한 화면에서 비교하기 위해 화면을 등분합니다. 행 선택 시 상하 2분할, 열 선택 시 좌우 2분할, 일반 셀 선택 시 최대 4분할됩니다.
차이점 창 나누기 분할 선은 마우스로 드래그해 위치 이동이 가능하고 더블 클릭하면 해제되지만, 틀 고정 선은 마우스로 움직일 수 없습니다.
교재 외 꿀팁: 정답 시트와 문제 시트 나란히 비교하기 보기 탭 → 새 창 클릭(동일 문서가 2번 창으로 열림) → 보기 탭 → 모두 정렬 → '세로' 선택. 왼쪽엔 내가 풀 시트, 오른쪽엔 정답 시트를 띄워 비교할 수 있습니다. 작업 후 한쪽 창을 닫으면 정상 복귀돼요.
시험 예시: 틀 고정 선을 마우스로 드래그해 위치를 옮길 수 있다 → 틀린 설명. 그건 창 나누기의 특징입니다.

정렬하기 전엔 반드시 숨기기부터 해제하세요.

그룹에 포함 안 된 다른 시트를 클릭해 그룹부터 해제한 뒤 작업하세요.

화면엔 "10000원"처럼 보여도, 문자를 직접 입력하면 더 이상 숫자가 아니라 문자 데이터가 되어 SUM 같은 함수 계산이 불가능해집니다.

값은 숫자 그대로 두고, 표시 형식에 #,##0"원"을 지정하세요.

자주 하는 실수

틀 고정과 창 나누기를 같은 기능으로 헷갈려요

둘 다 화면 보기를 나누지만, 틀 고정 선은 마우스로 옮길 수 없고 창 나누기 선은 드래그로 이동·더블 클릭으로 해제할 수 있습니다.

"틀 고정 = 고정, 창 나누기 = 이동 가능"으로 구분하세요.
